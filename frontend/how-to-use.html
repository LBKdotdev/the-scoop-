Overview

Three habits keep the app accurate: log production when you make a batch, do the nightly count before close, and check the dashboard for alerts before you start making ice cream.

Step 1 — Log Production

Every time you make a batch, log it right away.

  1. Tap the Production tab
  2. Enter your name or initials
  3. Find the flavor and set the quantity using + and −
  4. Tap Commit All

Takes about 5 seconds. Do it before you walk away from the batch.

Step 2 — Nightly Count

This replaces the paper tally sheet. Do it every night before close.

  1. Tap the Count tab
  2. Numbers are already pre-filled — that's the app's best guess based on last night's count + what you made − average consumption
  3. Walk the case and freezer. If the number is right, skip it. If it's wrong, tap + or − to fix it
  4. Filter by type (Tubs / Pints / Quarts) to move faster
  5. Tap Submit All Counts

Most numbers will already be close — you're mainly just confirming.

Step 3 — Check the Dashboard

The Dashboard shows everything at a glance.

  • Alerts — Red = almost out, yellow = running low. Make those first.
  • Morning Make List — What to make today based on last night's count and par levels
  • Current Inventory — Green = good, orange = getting low, red = out
  • Flavor Popularity — Best sellers for the week

Voice Input (Count Tab)

Instead of tapping +/− on every item, call them out by voice. Tap Voice Input and speak naturally.

One item at a time

"Vanilla pint four"
"Four pints of vanilla"
"Chocolate chip tub one"

Flavor, size, and number in any order — all three are needed.

Multiple items, mixed sizes

Use "and" between entries:

"Vanilla pint four and chocolate quart seven and chocolate chip tub one"

Multiple items, same size — fastest method

Say the size once, then list flavors:

"Tubs: nine chocolate chip, four vanilla, two strawberry"
"Pints: four vanilla, seven chocolate"

Set vs. Add

  • Saying a number → sets the count to that number
  • Saying "add", "also", "found", "plus", or "more" → adds to what's already there
"Found three vanilla pints" — bumps the count up by 3
"Vanilla pint three" — sets it to 3

After speaking, a confirmation screen shows what was parsed. Review it and tap Confirm. Say "undo" to reverse the last voice entry.

AI Insights

On the Dashboard, tap Generate Insights for smart suggestions — what to make tomorrow, which flavors are trending, if anything is being over-produced.

Managing Flavors

Add a flavor: Flavors tab → type the name → pick a category → Add Flavor.

Archive a flavor: Flavors tab → find it → tap Archive. Old data is kept, it just won't show up in counts anymore.

Stock Levels (Par): Set "Ready at open", "Make more at", and "One batch makes" for each flavor. The app uses these to build the morning make list.

Tips

  • You don't need to be perfect. Close counts are fine — the app gets smarter over time.
  • Log production right when you make it. Don't wait until end of day.
  • Check alerts before you start making ice cream. It tells you what's running low.
  • The app works on any phone. No app to download — just open it in your browser.
